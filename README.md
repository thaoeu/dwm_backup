	C - Ctrl

	S - Shift

<M-S- 数字键 >			将当前窗口移动到数字对应标签

<M-C- 数字键 >			使数字对应标签和当前标签同时显示

<M-0>					展示全部窗口

<M-S-q>					退出 Dwm
